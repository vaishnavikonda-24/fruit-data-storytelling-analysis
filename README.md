# 🍎 Fruit Data Storytelling & Statistical Validation

## 📌 Project Overview

This project was completed as part of a Data Analytics Internship. The objective was to analyze a fruit classification dataset, generate meaningful insights through data storytelling, and validate findings using statistical techniques.

## 🎯 Objectives

* Perform data analysis on a fruit dataset
* Calculate Key Performance Indicators (KPIs)
* Create data visualizations and dashboards
* Conduct hypothesis testing using a T-Test
* Generate business insights and recommendations

## 📂 Dataset Information

The dataset contains information about fruits, including:

* Weight
* Colour
* Label (Apple/Banana)
* Fruit Size
* Label Code

The dataset was cleaned and preprocessed before analysis.

## 📊 KPI Analysis

The following KPIs were calculated:

* Total Number of Fruits
* Number of Apples
* Number of Bananas
* Average Fruit Weight
* Maximum Fruit Weight
* Minimum Fruit Weight

## 📈 Visualizations Created

* Fruit Distribution Pie Chart
* Weight Distribution Histogram
* Average Weight by Fruit Type Bar Chart
* Fruit Size Distribution Chart

## 🔬 Statistical Validation

### Hypothesis Testing

**Null Hypothesis (H₀):**
There is no significant difference between the average weights of Apples and Bananas.

**Alternative Hypothesis (H₁):**
There is a significant difference between the average weights of Apples and Bananas.

**Method Used:**
Independent Two-Sample T-Test

The test was performed using Python and SciPy to validate analytical findings.
